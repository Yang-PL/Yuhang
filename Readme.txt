验证版本库

此文件夹是工作区，git文件夹中的index是暂存区，git add 文件名，就是将文件提交到暂存区。git commit -m “说明语句”是将暂存区的文件提交到工作区。git diff查看工作区和暂存区的差异，git diff --cached 查看暂存区和仓库差异。git add和git commit的反向命令git checkout和git reset HEAD 
git add的反向命令git checkout，撤销工作区修改，即把暂存区最新版本转移到工作区，git commit的反向命令git reset HEAD，就是把仓库最新版本转移到暂存区。
